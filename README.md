| Token   | Typ         | Przykład       | Opis                                                                             |
|---------|-------------|----------------|----------------------------------------------------------------------------------|
| KEYWORD | Operator    | if\|while\|def | Zarezerwowane słowa języka definiujące logikę (np. instrukcje warunkowe, pętle). |
| NUMBER  | Operator    | 123            | Dowolna sekwencja cyfr reprezentująca wartość liczbową (całkowitą).              |
| ID      | Operator    | zmienna        | Nazwy nadawane zmiennym, funkcjom lub klasom.                                    |
| PLUS    | Operator    | +              | Operator arytmetyczny dodawania.                                                 |
| MINUS   | Operator    | -              | Operator arytmetyczny odejmowania.                                               |
| MUL     | Operator    | *              | Operator arytmetyczny mnożenia.                                                  |
| DIV     | Operator    | /              | Operator arytmetyczny dzielenia.                                                 |
| ASSIGN  | Operator    | =              | Operator przypisania wartości do identyfikatora.                                 |
| LPAREN  | Nawias      | (              | Otwarcie nawiasu okrągłego (np. przy wywołaniu funkcji).                         |
| RPAREN  | Nawias      | )              | Zamknięcie nawiasu okrągłego.                                                    |
| LBRACE  | Klamra      | {              | Otwarcie klamry (często definiuje początek bloku kodu lub słownika).             |
| RBRACE  | Klamra      | }              | Zamknięcie klamry.                                                               |
| COLON   | Separator   | :              | Dwukropek, używany np. po definicji funkcji lub instrukcji if.                   |
| SPACE   | Białe znaki | \t\|\n         | Spacje, tabulatory i znaki nowej linii służące jako separatory.                  |
